# RivaTuner Statistics Server (RTSS)

## Função
Ferramenta de overlay e monitoramento em tempo real para Windows.

Permite:
- exibir overlay customizável durante jogos;
- monitorar uso de CPU, GPU, RAM, VRAM;
- exibir frametime e FPS;
- limitar framerate;
- integração com HWiNFO64 para dados avançados de sensores.

---

## Usar quando
- Benchmark de jogos com dados visuais;
- Monitorar uso de hardware em tempo real;
- Limitar FPS para reduzir consumo ou temperatura;
- Capturar dados de performance durante sessões.

---

## Recursos importantes
- overlay in-game customizável;
- framerate limiter;
- frametime graph;
- integração com HWiNFO64;
- suporte a múltiplos plugins;
- leve e portátil.

---

## Setup recomendado com HWiNFO64
1. Abrir HWiNFO64 em modo Sensors;
2. Ativar opção "Shared Memory Support" nas configurações do HWiNFO64;
3. Instalar plugin RTSS HWiNFO para o RTSS;
4. Configurar quais sensores exibir no overlay.

Esse combo é o padrão utilizado por criadores de conteúdo de benchmark.

---

## Cuidado
- Alguns jogos com anti-cheat podem detectar o RTSS como injeção;
- Overlay muito carregado pode afetar levemente a performance;
- Verificar compatibilidade com anti-cheat antes de usar online.

---

## Recomendações
- Combinar sempre com HWiNFO64 para dados completos;
- Manter overlay minimalista para não prejudicar FPS;
- Usar framerate limiter para controle térmico em sessões longas.

---

## Observações pessoais
Ferramenta essencial para benchmark visual no Windows.

Combinação HWiNFO64 + RTSS é o setup padrão da indústria para captura de dados de hardware durante jogos.

Independente do MSI Afterburner — pode ser usado sem ele.
