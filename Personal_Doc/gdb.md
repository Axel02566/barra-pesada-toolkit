# GDB

## Função
GNU Debugger.

Ferramenta avançada para:
- debugging;
- análise de memória;
- breakpoints;
- tracing;
- engenharia reversa.

---

## Usar quando
- Diagnosticar crash;
- Estudar software;
- Debugging avançado;
- Engenharia reversa;
- Exploração.

---

## Recursos importantes
- breakpoints;
- stack trace;
- memory inspection;
- registers;
- stepping;
- watchpoints.

---

## Comandos básicos
```bash
gdb programa
run
break main
next
step
backtrace
info registers
```

---

## Cuidado
- Curva de aprendizado alta.
- Alterações erradas podem causar comportamento inesperado.
- Malware deve ser analisado em VM isolada.

---

## Observações pessoais
Praticamente ferramenta obrigatória para:
- baixo nível;
- sistemas;
- engenharia reversa;
- desenvolvimento avançado.
