# Observações importantes

## Regras pessoais
- Sempre verificar hash de arquivos críticos;
- Nunca executar ferramenta desconhecida sem validação;
- Preferir ambiente isolado para testes perigosos;
- Manter backup dos dados importantes;
- Atualizar documentação periodicamente.

---

## Lembrete de atualização
Verificar atualizações disponíveis assim que for feita a instalação de qualquer ferramenta no sistema.

As versões armazenadas no pendrive são as mais recentes no momento do arquivamento, mas podem estar desatualizadas.

Priorizar atualização de:
- ferramentas de segurança;
- ferramentas com histórico de vulnerabilidades (ex: CCleaner);
- frameworks ativamente mantidos (ex: Metasploit, Ghidra, volatility3).

---

## Ferramentas de risco elevado
Usar com cautela:
- Metasploit
- BeEF
- Hashcat
- John the Ripper
- x64dbg
- volatility3
- Wireshark
- Nmap
