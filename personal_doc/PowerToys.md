# Microsoft PowerToys

## Função
Conjunto de utilitários avançados para Windows que expandem funcionalidades do sistema.

---

## Usar quando
- produtividade no Windows;
- automação leve;
- gerenciamento de janelas;
- renomeação em massa;
- busca rápida;
- ajustes de fluxo de trabalho.

---

## Recursos importantes

### FancyZones
- gerenciamento avançado de janelas.

### PowerRename
- renomeação em massa.

### PowerToys Run
- launcher rápido (tipo Spotlight).

### Keyboard Manager
- remapeamento de teclas.

### File Explorer Add-ons
- preview de arquivos extras.

### Image Resizer
- redimensionamento rápido.

---

## Cuidado
- algumas funções interferem com atalhos padrão do Windows;
- pode conflitar com softwares de terceiros;
- exige adaptação inicial.

---

## Recomendações
- configurar FancyZones para multitarefa;
- usar PowerToys Run como substituto do menu iniciar;
- manter módulos que você realmente usa ativos.

---

## Observações pessoais
Ferramenta extremamente útil para:
- produtividade;
- organização;
- fluxo de trabalho técnico;
- uso diário avançado.

Quase um “upgrade do Windows sem modificar o sistema”.
