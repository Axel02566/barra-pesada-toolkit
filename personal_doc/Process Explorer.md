# Process Explorer

## Função
Gerenciador avançado de processos da Sysinternals.

Muito mais poderoso que o Task Manager.

Permite visualizar:
- árvore de processos;
- DLLs;
- handles;
- consumo;
- threads;
- assinaturas digitais;
- serviços associados.

---

## Usar quando
- Diagnosticar processo suspeito;
- Encontrar vazamento de memória;
- Verificar uso anormal;
- Identificar malware;
- Descobrir arquivo bloqueado.

---

## Recursos importantes
- Verify Signatures;
- VirusTotal integration;
- Handle search;
- árvore hierárquica;
- análise de DLLs.

---

## Cuidado
- Finalizar processo errado pode travar o sistema.
- Alguns processos críticos reiniciam automaticamente.
- Malware pode tentar se esconder.

---

## Observações pessoais
Uma das ferramentas mais úteis já feitas para Windows.

Excelente para:
- troubleshooting;
- análise de comportamento;
- diagnóstico profundo.
