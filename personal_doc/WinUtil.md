# WinUtil (Chris Titus Tech's Windows Utility)

## Função
Utilitário open-source para configuração, debloat e otimização do Windows.

Permite:
- instalar programas em massa via winget;
- remover bloatware;
- aplicar tweaks de performance e privacidade;
- configurar o Windows após instalação limpa;
- reparar componentes do sistema.

---

## Usar quando
- Configurar Windows recém instalado;
- Remover aplicativos desnecessários;
- Aplicar tweaks de privacidade e performance;
- Instalar programas em lote rapidamente;
- Reparar componentes do sistema.

---

## Recursos importantes
- instalação de programas via winget;
- debloat automatizado;
- tweaks de privacidade;
- configuração de DNS;
- MicroWin (criação de ISO customizada do Windows);
- reparo do sistema integrado.

---

## Como executar
Direto via PowerShell sem instalação:
```powershell
irm christitus.com/win | iex
```

Ou via arquivo local:
```powershell
.\winutil.ps1
```

---

## Cuidado
- Alguns tweaks podem quebrar funcionalidades do Windows.
- MicroWin remove componentes que podem ser necessários.
- Aplicar mudanças gradualmente.
- Criar restore point antes de usar.

---

## Recomendações
- Criar restore point antes de qualquer alteração;
- Revisar cada opção antes de aplicar;
- Não aplicar todos os tweaks de uma vez;
- Testar em ambiente não crítico primeiro.
